Мой первый бот "Давай поищем в Википедии!"

Список доступных команд:
/start - запуск бота
/info, /help - описание бота
/search - запуск поиска в Википедии

Для начала поиска в Википедии необходимо ввести команду /search, выбрать категорию информации и далее сам элемент.
В сообщении о выборе категории и соответствующего ей элемента встроены inline-меню.
Выбор элемента из категории поиска реализован с помощью механизма "цепочки сообщений".

Если ничего не будет найдено, то появится сообщение 'Ничего не найдено. Повторим?'

Важно!
Поскольку поиск осуществляется в Википедии и бот расположен на домашнем компьютере, то могут возникать временные задержки.
В случае их ощутимой длительности рекомендуется ввести поисковое сообщение повторно.
